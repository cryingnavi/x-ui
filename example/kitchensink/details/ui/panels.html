What's View ?
모바일에 최적화 되어 있는 UI 를 생성하여 줍니다. 기존 다른 UI Framework 와는 다르게 최소한의 학습만으로 사용할 수 있으며 디자인/퍼블리셔는 자신들의 작업을 방법을 그대로 유지할 수 있습니다.
View 다음과 같은 기능을 가지고 있다.
  • 멀티 해상도, 멀티 디바이스에 대응한다.(flexible)
  • 모바일에 최적화된 스크롤 기능을 내장하고 있다.(iScroll)
  • 어떠한 형태의 컨텐츠든지 표현할 수 있으며 레이아웃을 보장한다.
  • View간의 관계 설정을 통하여 다양한 레이아웃을 쉽고 간단하게 만들 수 있다.
  • ViewController 의 제공으로 하위 View 들에 대한 화면 전환을 제공한다. 이때 애니메이션을 사용할 수 있다.
  • View 를 팝업으로 생성할 수 있다.(floating)
  • 좌측/우측의 숨겨진 메뉴를 쉽게 생성할 수 있다.(Panels)
