X-UI

( Rich UI Mobile Client )

What's X-UI ?
모바일에 최적화 되어 있는 UI 를 생성하여 줍니다. 기존 다른 UI Framework 와는 다르게 최소한의 학습만으로 사용할 수 있으며 디자인/퍼블리셔는 자신들의 작업 방법을 그대로 유지할 수 있습니다.
장점
  • 적은 학습으로 뛰어난 User Interface 를 구현할 수 있다.
  • 높은 자유도로 퍼블리셔의 기존 작업 방법을 해치지 않는다.
  • 화면 전환 방법을 제공하며 전환시 애니메이션을 제공한다.
  • 다양한 MVC Framework 와 어우러져 사용할 수 있다.
  • 모바일 웹앱 어플리케이션 제작에 최적화 되어있다.
  • Drag and Drop 기능 제공
