# x-ui
HTML 5 기반의 작은 ui framework 입니다. 웹 어플리케이션 제작 유용하도록 설계되었으며 리치한 UI 를 구현하는데 필요한 최소한의 기능을 제공합니다.
x-ui 는 크게 다음과 같이 구성되어 있습니다.

1. core
2. util
3. widget
