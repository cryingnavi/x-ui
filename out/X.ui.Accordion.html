Class: Accordion

X.ui. Accordion

X.ui.Accordion 클래스는 Accordion ui 를 생성한다.

new Accordion()

Properties:
Name Type Description
config.titles Array 각 탭의 title 을 설정한다.
config.items Array 각 탭의 view를 설정한다.
config.activeIndex Number 초기 open될 탭을 설정한다. Default : 0
Source:
  • x-ui.js
Example
var accordion = new X.ui.Accordion({
     titles: ["Tab 1", "Tab 2", "Tab 3"],
     activeIndex: 0,
     items: [new X.View(), new X.View(), new X.View()]
});
accordion.render();
