Class: View

X. View

View 를 생성한다. View란 화면의 기본 단위이며 기본적인 container 역할을 수행한다.

new View()

Properties:
Name Type Description
el String | jQuery | HtmlElement view 를 생성할 엘리먼트를 지정한다.
width Number | String view 의 가로 사이즈를 지정한다. autoSize 가 true 일 경우 무시된다.
height Number | String view 의 세로 사이즈를 지정한다. autoSize 가 true 일 경우 무시된다.
autoSize Boolean 자동으로 부모 요소를 가득 채울지를 결정한다. 기본값이 True 이다
scroll Boolean 스크롤을 생성할지 여부를 지정한다.
scrollConfig Object 스크롤 options 을 지정한다.
content String | jQuery view 가 가질 content 를 지정한다. html 문자열 또는 jquery 객체를 받는다.
children Array 하위에 포함할 다른 controll 들을 지정한다.
toolbars Array view 가 포함할 툴바를 지정한다.
floating Boolean view 를 팝업형태로 화면에 띄울지를 지정한다.
overlay Boolean floating true 일때 회색 배경을 덮을지를 지정한다.
viewController X.util.ViewController
layout String 하위 view 들이 가로 또는 세로로 배열되도록 한다. 기본값은 세로이다. 'x' | 'y'.
flexible Boolean view 가 flexible 하게 자신의 크기를 결정할지 여부이다. false 로 지정하고 width, height 를 지정하면 해당 크기대로 view 가 결정된다.
panels Object 왼쪽, 오른쪽에 숨겨진 view를 지정한다.
className String view 에 지정할 클래스이름
style String | Object view 에 적용할 style

false 일 경우 view가 명시적인 크기를 가져야하며 해당 크기로 view 가 생성된다.
