Class: Spinner

X.ui. Spinner

spinner 를 생성한다.

new Spinner()

Properties:
Name Type Description
min Number 최소값을 지정한다. Default: 0
max Number 최고값을 지정한다. Default: 100
step Number step을 지정한다. Default: 1
defaultValue Number 기본값을 지정한다. Default: 0
Source:
  • x-ui.js
Example
var spinner = new X.ui.Spinner({
     min: 0,
		max: 100,
		step: 1,
		defaultValue: 0
});
spinner.render();

<div data-role="spinner"></div>
