Class: Slider

X.ui. Slider

슬라이더바를 생성한다.

new Slider()

Properties:
Name Type Description
min Number 슬라이더의 최소값을 지정한다. Default: 0
max Number 슬라이더의 최고값을 지정한다. Default: 100
한번에 Number 이동할 슬라이더의 step을 지정한다. Default: 1
defaultValue Number 슬라이더의 기본값을 지정한다. Default: 50
direction String 'x' 나 'y'. 슬라이더의 방향을 지정한다. Default: 'x'
subhandle Boolean 두개의 핸들을 생성할지 여부를 지정한다. Default: false
Source:
  • x-ui.js
Example
var slider = new X.ui.Slider({
     min: 0,
		max: 100,
		step: 1,
		defaultValue: 50,
		direction: 'x',
		subhandle: false
});
slider.render();


<div data-ui="slider" style="width: 300px;margin:30px;" data-default-value="25"></div>
