Class: Draggable

X.util. Draggable

X.util.Draggable 클래스는 엘리먼트를 드래그 가능 엘리먼트로 생성한다. both or x or y 셋 중 하나의 값을 가지며 각각 양방향, 가로, 세로 방향으로 움직이게끔 지정한다.

new Draggable()

Properties:
Name Type Description
config.constrain Boolen 드래그가 가능한 범위를 설정한다. 셀렉터로 설정가능하며 해당 셀렉터로 검색된 엘리먼트를 범위로 갖는다. 기본값은 false 이다. false 일 경우 범위가 존재하지 않는다.
config.handle Object 드래그 객체 내의 특정 영역을 handle 로 지정한다. 지정할 시 해당 영역을 클릭하여야 드래그가 시작된다. 기본값은 null 이다.
config.revert Boolen 드래그가 끝났을 경우 원래 위치로 되돌아 갈지 여부이다. 기본값은 false 이다.
config.revertDuration Number rever가 true 시 애니메이션 속도를 지정한다. 기본값은 200 이다. 0 지정시 애니메이션이 눈에 보이지 않는다.

var drag = new X.util.Draggable({
