Class: SwitchBox

X.ui. SwitchBox

SwitchBox 를 생성한다.

new SwitchBox()

Properties:
Name Type Description
checked Boolean 체크 상태를 지정한다. Default: false
on String on 상태의 텍스트를 지정한다. Default: 'ON'
off String off 상태의 텍스트를 지정한다. Default: 'OFF'
Source:
  • x-ui.js
Example
var sb = new X.ui.SwitchBox({
     checked: false,
		on: 'ON',
		off: 'OFF'
});
sb.render();

<div data-ui="switchbox"></div>
