Class: LayoutView

X.ui. LayoutView

Form 컴포넌트를 묶어 하나의 View를 생성한다.

new LayoutView()

Properties:
Name Type Description
minSize Object 각 west, east, south, nouth 의 최소 사이즈를 지정한다.
maxSize Object 각 west, east, south, nouth 의 최고 사이즈를 지정한다.
size Object 각 west, east, south, nouth 의 초기 사이즈를 지정한다.
regions Object 각 west, east, south, nouth 에 해당하는 view를 지정한다.
Source:
  • x-ui.js
Example
var layoutview = new X.ui.LayoutView({
		autoSize: true,
		maxSize: { west: 700, east: 700, south: 100, nouth: 100 },
     minSize: { west: 200, east: 200, south: 50, nouth: 50 }, 
     size: { west: 300, east: 300, south: 150, nouth: 150 }, 
		regions : {
		    south: new X.View(),
			nouth: new X.View(),
			west: new X.View(),
			center: new X.View(),
			east: new X.View()
		},
		listener: {
			resize: function(){
				
			}
     }
});
layoutview.render();
