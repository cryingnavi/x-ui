Class: Carousel

X.ui. Carousel

X.ui.Carousel 클래스는 Carousel ui 를 생성한다.

new Carousel()

Properties:
Name Type Description
config.direction String carousel 의 이동방향을 지정한다. x 또는 y 로 가로 세로 방향을 지정한다. Default: 'x'
config.activeIndex Number 초기 활성화될 아이템을 지정한다. Defautl: 0.
config.duration Number 애니메이션 이동 속도를 지정한다. Defautl: 200.
config.children Array 각 탭의 view를 설정한다.
Source:
  • x-ui.js
Example
var carousel = new X.ui.Carousel({
     direction: "x",
     activeIndex: 0,
     duration: 200,
     children: [new X.View(), new X.View(), new X.View()]
});
carousel.render();
