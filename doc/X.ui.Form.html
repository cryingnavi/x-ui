Class: Form

X.ui. Form

form 요소들의 Base 클래스 이다.

new Form()

Properties:
Name Type Description
config.name String 넘어온 문자열을 키로 삼아 FormView에서 serialize 하게 된다. name을 넘기지 않을 경우 id 를 사용한다.
config.disabled Boolean 폼요소의 비활성화 상태를 나타낸다.
config.defaultValue String | Number 폼요소의 초기값을 설정한다.
config.label String 폼 요소의 label을 설정한다.
style Object | String 엘리먼트에 적용할 인라인 스타일을 지정한다.
