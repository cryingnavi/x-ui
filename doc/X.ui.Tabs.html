Class: Tabs

X.ui. Tabs

Tab 형태의 화면을 생성한다.

new Tabs()

Properties:
Name Type Description
position String 'top' 나 'bottom'. 탭바의 위치를 지정한다. Default: 'top'
activeIndex Number on 기본으로 화면에 나타날 하위 view의 인덱스를 지정한다. Default: 0
titles Array 탭바의 각 탭 요소의 타이틀을 지정한다.
transition String 애니메이션 종류를 지정한다. viewcontroller의 애니메이션과 종류가 같다. Default: 'slide'
children Array 각 탭에 들어갈 view를 지정한다.
Source:
  • x-ui.js
Example
var tabs = new X.ui.Tabs({
     position: 'top',
		activeIndex: 0,
		titles: ['Tabs 1', 'Tabs 2', 'Tabs 3'],
		transition: 'slide',
		children: [
         new X.View(), new X.View(), new X.View()
     ]
});
tabs.render();

<div data-ui="tabs">
		<div data-ui="view" data-title="Tabs 1" style="background:red;">Tabs 1</div>
		<div data-ui="view" data-title="Tabs 2" style="background:blue;">Tabs 2</div>
		<div data-ui="view" data-title="Tabs 3" style="background:yellow;">Tabs 3</div>
	</div>
