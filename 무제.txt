x-ui

X-UI Framework 는 모바일 환경에 특화된 Javascript User Interface Framework 이다. jquery 기반 하에 작성되었으며 0.0.0 이상의 버젼에 맞춰져 있다.


x-ui 는 다음과 같은 장점을 지닌다
- 높은 자유도
- 간결하고 쉬운 소스 구조
- 다양한 widget 포함
- 멀티해상도, 멀티디바이스, 멀티OS 대응
(현재 버젼은 webkit 계열 브라우저에서만 가능하나 로드맵에 따라 타브라우저도 대응할 것임)


x-ui 의 구성요소
- core
- util
- View
- ui
